# Factorial (bài toán giai thừa) 

## Các bước tính toán của Factorial
```
factorial(4)
 └─ factorial(3)
     └─ factorial(2)
         └─ factorial(1)
             └─ factorial(0)
```

## Độ phức tạp của thuật toán
Time Complexity: O(n)

## Điều kiện dừng của thuật toán
```
if (n == 0) return 1;
```
