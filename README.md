## How to start and testing project
### Setup project
1. Install python from https://www.python.org/
2. Install extension for python in vscode.
3. Ctrl + Shift + P -> Python: Select Interpreter -> Choose python.exe.

### Run project
1. run ```python [filename].py``` in terminal:
```
python [filepath]\[filename].py
```
